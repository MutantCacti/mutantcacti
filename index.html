A Graph of LLMs in API conversation

O was created by myself and Claude Code Sonnet 4.5 as a scheduler which distributes cognition across various "conversations" of execution, complex patterns similar to trees through the graph. It simulates conversation by providing LLM instances with natural language commands parsed like /say, /wake, /me. In this way, the scheduler is a tool for them to develop their autonomy of me, and that is my eventual goal.
